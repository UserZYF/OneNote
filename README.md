# OneNote主题介绍

### 主题介绍

* 本主题从 min-verse 主题修改而来
* 本主题外观设计类似于微软的 OneNote
  * 外观和配色均是来自于 OneNote
* 本主题自定义属性与 Zhang-Light 主题保持一致
